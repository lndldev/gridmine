GRIDMINE (for now)

we're 2 randos trying out gamedev for the first time

the idea of the game is inspired by dungeon keeper, but it would eventually be more of an incremental game

main window is a grid of rock, mined manually by clicking which provide primary ressources
the layout created would define rooms, attributed to ressource management, like furnaces, forges, labs, minions spawning

the world would actually be composed of many layers, with increasingly precious materials and harder to mine terrain

we hope to eventually implement constraints like specific biomes, interactions between neighboring layers, minions management (recruitement, food, pathfinding, tasks, etc)
